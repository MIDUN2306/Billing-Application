## Color Scheme

- **Orange** (Raw Materials): Warm, represents raw ingredients
- **Red** (Total Costs): Alert color for money going out
- **Purple** (Petty Cash): Distinct color for cash management
- **Gray** (Purchases): Neutral, informational only
- **Green** (Sales): Positive, money coming in
- **Yellow** (Pending): Warning, needs attention

## User Flow

1. User lands on dashboard
2. Sees 4 primary cost cards at top
3. Clicks on any colored card (Orange, Red, or Purple)
4. Modal opens with detailed table
5. Reviews transactions and totals
6. Closes modal to return to dashboard
