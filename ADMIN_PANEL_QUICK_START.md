# Admin Panel - Quick Start Guide

## Accessing the Admin Panel

1. Log in with an admin account
2. Click "Admin Panel" in the sidebar (shield icon)
3. You'll see three tabs: Stores, Users, and Role Management

## Quick Actions

### Create a New Store

1. Go to **Stores** tab
2. Click **"Create Store"** button
3. Fill in:
   - Store Name (required)
   - Owner Name, Phone, Email (optional)
   - Address, GST Number (optional)
4. Click **"Create Store"**

### Create a New User

1. Go to **Users** tab
2. Click **"Create User"** button
3. Fill in:
   - Full Name (required)
   - Email (required)
   - Password (required, min 6 characters)
   - Phone (optional)
   - Role: Admin, Manager, or Staff
   - Store: Select from dropdown
4. Click **"Create User"**

**Important:** Users need to be assigned to a store to access store-specific features!

### Transfer User to Another Store

1. Go to **Users** tab
2. Find the user you want to transfer
3. Click the **transfer icon** (arrows)
4. Select the target store
5. Click **"Transfer User"**

The user will immediately lose access to the old store and gain access to the new store.

### Change User Role

1. Go to **Users** tab
2. Find the user
3. Click the **edit icon** (pencil)
4. Change the **Role** dropdown
5. Click **"Save Changes"**

### Deactivate a User

1. Go to **Users** tab
2. Find the user
3. Click the **edit icon** (pencil)
4. Click **"Deactivate"** button
5. User can no longer log in

To reactivate, click **"Activate"** in the same location.

## Role Permissions

### Admin (Full Access)
- Create stores and users
- Manage all system settings
- Access all store data
- Transfer users between stores

### Manager (Store Management)
- Manage products and inventory
- Process sales and purchases
- Record expenses
- View reports
- Cannot create users or stores

### Staff (Basic Operations)
- Process sales (POS)
- View products and inventory
- View sales history
- Limited access to reports

## Tips

- **Search**: Use the search bar to quickly find stores or users
- **Filters**: Filter users by role or store for easier management
- **Mobile**: The admin panel works great on mobile devices too!
- **Status**: Green = Active, Gray = Inactive
- **Transfer Warning**: Users lose access to old store data immediately upon transfer

## Common Workflows

### Setting Up a New Store

1. Create the store (Stores tab)
2. Create a manager user for that store (Users tab)
3. Create staff users for that store (Users tab)
4. Manager can now log in and set up products

### Moving an Employee

1. Go to Users tab
2. Click transfer icon on the user
3. Select new store
4. Confirm transfer
5. Done! User now has access to new store

### Promoting a User

1. Go to Users tab
2. Click edit icon on the user
3. Change role from Staff → Manager or Manager → Admin
4. Save changes
5. User now has elevated permissions

## Need Help?

Check the **Role Management** tab to see detailed permissions for each role.
