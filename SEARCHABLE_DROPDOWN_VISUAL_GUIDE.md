## Tips for Users

### Tip 1: Use SKU for Speed
If you have the SKU code from your supplier invoice:
- Just type the SKU number
- Select the matching product
- Much faster than scrolling!

### Tip 2: Partial Names Work
You don't need to type the full name:
- "bis" finds "Biscuits"
- "sam" finds "Samosa Large"
- "bun" finds all buns

### Tip 3: Filter by Type
Need to see only ingredients?
- Type "making"
- See only the 4 making products

Need to see only ready-to-use?
- Type "ready"
- See all 36 ready-to-use products

### Tip 4: Click Outside to Close
Don't want to select anything?
- Just click outside the dropdown
- It will close automatically
