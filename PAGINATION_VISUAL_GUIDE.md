## Technical Details

### Page Size: 20
- Small enough for fast loading
- Large enough to reduce clicks
- Optimal for most screen sizes

### Query Strategy
```sql
-- Total query (once)
SELECT SUM(total_cost), COUNT(*) 
FROM table WHERE conditions

-- Page query (per page)
SELECT * FROM table 
WHERE conditions 
ORDER BY date DESC 
LIMIT 20 OFFSET (page-1)*20
```
