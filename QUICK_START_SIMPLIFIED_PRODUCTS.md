## âš¡ Quick Tips

### 1. Product Types
- **ðŸ“¦ Manufactured** = Made from raw materials (Tea, Coffee, Juice)
- **ðŸ›’ Simple** = Bought ready-made (Biscuits, Chips, Samosas)

### 2. Recipe Yield
- This is how many units your recipe makes
- Example: "2L milk + tea powder = 50 cups"
- System auto-calculates for larger batches

### 3. Stock Warnings
- Yellow alerts show if stock is low
- Can't save if insufficient stock
- Add more raw materials first

### 4. SKU Generation
- Click "Generate" for auto SKU
- Format: First 3 letters + random number
- Example: "TEA-1234"

### 5. Ingredients
- Only shows raw materials with stock > 0
- Can add unlimited ingredients
- Can't add same ingredient twice
- Unit auto-fills from raw material

## ðŸ” Troubleshooting

### Problem: "Insufficient stock" error
**Solution**: 
1. Go to Raw Materials page
2. Add more stock for the needed material
3. Come back and try again

### Problem: Can't add ingredient
**Solution**: 
- Check if raw material has stock > 0
- If not, add stock first

### Problem: Duplicate ingredient error
**Solution**: 
- Click trash icon to remove old one
- Then add the new one

### Problem: Recipe yield required
**Solution**: 
- Enter how many units your recipe makes
- Example: "50 cups" or "20 glasses"
